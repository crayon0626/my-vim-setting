## Installation
1. Clone the repository
 ```Shell
 git clone https://github.com/crayon0626/my-vim-setting.git
 ```

2. Clone the vundle
 ```Shell
 git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim 
 ```

3. Install Plugins
 ```Shell
 vim
 :PluginInstall
 ```
